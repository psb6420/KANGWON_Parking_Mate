# 15099883 전체 주차장 지도

공공데이터포털 `한국교통안전공단_주차정보 제공 API` 중 `PrkSttusInfo`만 호출해서 지도에 표시합니다.

## 사용 API

```text
https://apis.data.go.kr/B553881/Parking/PrkSttusInfo
```

화면에서는 서비스키만 입력합니다. `pageNo`, `numOfRows`, `format=2`는 서버가 내부에서 자동으로 넣습니다.

```text
serviceKey = 사용자가 입력
numOfRows = 5000
pageNo = 1부터 마지막 페이지까지 자동 순회
format = 2
```

## 실행

```powershell
cd D:\X+AI+SW\kakao-map-app
node server.js
```

브라우저에서 `http://localhost:8080/`을 엽니다.

## 화면 동작

1. 공공데이터포털 서비스키를 입력합니다.
2. `전체 불러오기`를 누릅니다.
3. 서버가 `PrkSttusInfo` 전체 페이지를 순서대로 호출합니다.
4. 호출 완료 후 좌표가 있는 주차장을 지도에 표시합니다.
5. 주소 필드에 `강원`이 들어간 데이터가 나온 원본 페이지 번호를 화면에 출력합니다.
6. 일부 페이지 호출이 실패하면 실패 페이지 번호도 같이 출력하고, 나머지 페이지 수집은 계속 진행합니다.

전체 데이터는 100만 건 이상이라 전체 호출에 시간이 걸릴 수 있습니다. 서버는 전체 페이지를 호출하고, 강원 전체 마커를 표시할 수 있도록 응답 목록 제한을 기본 100000건으로 둡니다. 이 제한은 `PARKING_MAP_ROW_LIMIT` 환경 변수로 조정할 수 있습니다.
